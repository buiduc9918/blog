Bước 1: Tạo file JS điều hành ứng dụng
- Định nghĩa các biến cơ bản:
  + const: hằng số dùng để khai báo thư viện
  + port: cổng chạy ứng dụng
  + express: thư viện backend chính
  + node: môi trường chạy JavaScript phía server
- Tạo file `package.json` để khai báo thư viện và phụ thuộc
- `package-lock.json` lưu cấu hình phiên bản phụ thuộc chính xác

Bước 2: Cài đặt thư viện cơ bản
- Cài Express:
  npm i express
- Thêm `path` để xử lý đường dẫn file
- Cài các thư viện hỗ trợ:
  + morgan: ghi log HTTP
  + express-handlebars: template engine
  + nodemon: tự động reload khi lưu code
  + sass: biên dịch SCSS sang CSS
  + prettier: định dạng code
  + lint-staged: chạy prettier trước khi commit
  + husky: chạy lint-staged khi push/commit
  + http-errors, jade: debug và xử lý lỗi
  + mongoose: kết nối database với model/controller

Lưu ý khi cài đặt:
- Phân loại gói cho môi trường khác nhau:
  + dependencies: thư viện dùng ở production
  + devDependencies: công cụ chỉ dùng khi phát triển

Bước 3: Tổ chức cấu trúc thư mục theo `src`
Trong `src` nên có:
- `app/`: cấu hình và khởi tạo ứng dụng, kết nối database
- `config/`: cấu hình database và môi trường
- `public/`: chứa file tĩnh như CSS, ảnh, JavaScript client
- `resource/`: chứa file giao diện
- `routes/`: định nghĩa các đường dẫn

Bước 4: Phân chia chức năng MVC
- `app`:
  + `controllers/`: xử lý các action theo route
  + `model/`: định nghĩa cấu trúc dữ liệu và truy vấn database
- `config`:
  + cấu hình database và môi trường ứng dụng
- `public`:
  + `css/`: lưu file giao diện CSS
  + `image/`, `sound/`: lưu ảnh, âm thanh và tài nguyên tĩnh khác
- `resource`:
  + `scss/`: chứa file SCSS và biến giao diện
  + `views/`: chứa template view và layout
    * layout chính (`main`) chứa phần header/footer và body
    * template con sẽ được chèn vào `{{{body}}}`

Ví dụ cấu trúc layout Handlebars:
  {{> header}}
  <div class="container">
    {{{body}}}
  </div>
  {{> footer}}

Với Bootstrap, thêm các script cần thiết vào layout:
  <script src="https://cdn.jsdelivr.net/npm/jquery@3.5.1/dist/jquery.slim.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/popper.js@1.16.1/dist/umd/popper.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/js/bootstrap.min.js"></script>

Bước 5: Hiểu cách hoạt động của web và routing
- Hiểu `req`, `res` trong HTTP
- Hiểu cách dữ liệu chạy giữa client và server
- Ví dụ: `http://localhost/news?abc=...` dùng query để tìm kiếm

Bước 6: Thiết lập mô hình MVC
- Model View Controller:
  + Controller giao tiếp với Browser qua router và dispatcher
  + Router định nghĩa đường dẫn đến action
- Cấu trúc ứng dụng:
  + `src/index.js`: điểm vào chính của server
  + `config/db/`: cấu hình kết nối database
  + `resource/` và `public/`: chứa view và file tĩnh
  + `routes/`: điều hướng các link của web
  + `app/`: điều hướng dữ liệu, lấy dữ liệu từ DB và render view
  + `src/app/controllers`: chứa controller xử lý logic

Bước 8: Ví dụ `src/index.js`
const path = require('path');
const express = require('express');
const morgan = require('morgan');
const handlebars = require('express-handlebars');
const app = express();
const port = 3000;
const route = require('./routes/index');
const db = require('./config/db');

// Connect to DB
db.connect();

app.use(express.static(path.join(__dirname, 'public')));
app.use(express.urlencoded({ extended: true }));
app.use(express.json());
app.use(morgan('combined'));
app.engine('.hbs', handlebars.engine({ extname: '.hbs' }));
app.set('view engine', 'hbs');
app.set('views', path.join(__dirname, 'resource/views'));

// Route init
route(app);

app.listen(port, () => {
  console.log(`Example app listening on port http://localhost:${port}`);
});
